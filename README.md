# Fred Feng · CV

> 🔗 **Live Site:** [fred-cv.vercel.app](https://fred-cv.vercel.app/)

A personal CV web app built with React + Vite, deployed on Vercel.

## Features

- **Intro Page** — Animated particle network background, staggered text reveal, ticker bar
- **CV Page** — Two-column layout with sidebar (skills, education, contact) and main content (profile, open source projects, work experience)
- Smooth page transitions, hover interactions, dark tech aesthetic

## Tech Stack

- React 18
- Vite 4
- CSS animations (no external animation library)
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build    # outputs to dist/
npm run preview  # preview production build locally
```

Push to GitHub — Vercel auto-deploys on every commit.
